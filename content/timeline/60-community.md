## Community — code for the congregation

Away from paid work, I build for my church. The main thing is a **Gesangbuch
PWA** — a German hymnal as a progressive web app — made so the congregation can
actually *use* it, on a phone, in a pew, offline. It runs in German and in
English, the two languages I speak, so nobody is left squinting at the wrong page.

The code is open source on my GitHub (Blaxzter), because something the community
relies on shouldn't live only on my laptop. It gets the same engineering care I
give a client — only here the payment is that people sing from it. Building for a
community, not a contract, is its own kind of through-line: the work is the same,
the gift is the difference.
